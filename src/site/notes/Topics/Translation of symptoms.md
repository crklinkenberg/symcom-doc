- Symptoms when imported to particular source, is imported using a language(either German or English).
    
- The language in which the source has symptoms are listed in the import source page, each source are listed in rows. Symptoms can be added here with new language, if the source does not contain symptoms with that language.
    
- Translation icon can be clicked in the comparison page to view the symptom text in German or English language.
    
- Translations of all symptoms are loaded on page load and kept in an jQuery array. This is then manipulated when open all translation or just translation button is clicked.
