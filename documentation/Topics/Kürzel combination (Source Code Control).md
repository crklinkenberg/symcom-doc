Sorce code is made combining abbreviations of auhor + source title abbreviation 3 + band + year of the source.

For example: Source code can be : Jr SK2 1843.

Here "Jr" is author abbreviation, "SK" is title abbreviation of  the source, "2" is band and "1843" is the year.

  

Technical Details:

PHP function getQuelleAbbreviationForMainSection() controls the source code display in the main section.

  

PHP function getSourceAbbreviation() controls the source code display in the symcom api section.

  

Related sql tables:

- autor
    
- quelle
